# PIXEL COLORING

* ## DESCRIPTION

You play as an arrow, moving it across a grid field and painting over squares.

* ## FUNCTIONS

single player mode

control:

-  up(`⭡`)
-  down(`⭣`)
-  right(`⭢`)
-  left(`⭠`)

- paint over(`enter`)
- close programm(`space`)
